Dziennik pojazdów dla OSP napisany według konwencji Repository Pattern z oddzieloną optymalizacją pod PWA.

## Szybki start lokalnie

```bash
python -m venv .venv
# source .venv/bin/activate (macOS/Linux)
# .\.venv\Scripts\Activate.ps1 (Windows)
pip install -r requirements.txt

# uruchamianie w trybie dev (auto-reload)
python run.py
```
